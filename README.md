# CLStatusBarHUD
簡單易用的指示器小框架

## 顯示成功訊息
```objc
[CLStatusBarHUD showSuccess:@"加載成功!"];
```

## 顯示失敗訊息
```objc
[CLStatusBarHUD showError:@"加載失敗!"];
```

## 顯示加載訊息
```objc
[CLStatusBarHUD showLoading:@"正在加載中..."];
```

## 隱藏
```objc
[CLStatusBarHUD hide];
```

## 顯示普通訊息
```objc
[CLStatusBarHUD showMessage:@"没有什麼事!!!!"];
```
